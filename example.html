How is $.energysaver() saving energy?


Energysaver(a jquery plugin) was created by @motyar to remind us all of the need to take small steps in our everyday lives to save energy.

$.energysaver() saves energy because the screen background goes black if no event occurs for a given period.

"A given monitor requires more power to display a white (or light) screen than a black (or dark) screen."

 
//usage
$.energysaver();
//or - 
$.energysaver({options});
//default options - 
{
    delay: 5000,
    events: 'mousemove mousedown keydown'
}
